This project focuses on analyzing India’s export performance to Oceanic countries using Power BI. The objective is to transform raw data into meaningful insights through interactive dashboards and visualizations.
The dataset has been sourced from the Indian Data Portal and includes detailed information on export value, quantity, commodities, countries, and regions.

Objectives were to
Analyze export trends over time
Identify top-performing countries and commodities
Compare export value and quantity
Understand regional contribution to exports
Build an interactive and user-friendly Power BI dashboard

Key Insights that found from the project :

Exports are highly concentrated in a few countries, especially major markets in the Oceanic region

A small number of commodities contribute to the major share of export value

Export trends show seasonal fluctuations rather than consistent growth

The majority of exports are measured in kilograms, indicating bulk commodity trade

There exists a mix of high-volume low-value and low-volume high-value commodities

Some regions and countries show low participation, indicating potential growth opportunities
